## 安装方法

1. 安装浏览器插件 [Tampermonkey](https://www.tampermonkey.net/)。
2. 打开 [Greasy Fork 发布页](https://greasyfork.org/zh-CN/scripts/565802)
3. 点击安装按钮，确认安装。
4. 刷新 GitHub 页面，享受清爽的阅读体验。
